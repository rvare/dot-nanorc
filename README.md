# Nanorc

This repository is used to keep track of my `.nanorc` file. Anyone else is also free to use it for their own systems as well.

# How to use the file

Clone the repo where it is most convient for you.

To use the file, copy the `nanorc` file to your home directory and rename it to `.nanorc`. (Basically the same name but a dot `.` prepended.)
